# Readability.jl

Julia package that analyzes the difficulty of texts using different indexes like Gunning-Fog, SMOG, and Flesch-Kincaid.

## Installation

use this command in the Julia REPL:

```julia
using Pkg; Pkg.add("Readability")
```
